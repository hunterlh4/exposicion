## Introducción
La inteligencia artificial (IA) se utiliza hoy en día en una gran variedad de áreas: diseño de interfaces, programación, creación de música, arte digital, productividad, automatización y mucho más. Estas tecnologías están transformando la manera en que trabajamos. En esta presentación conocerás herramientas de IA aplicadas a diferentes categorías.

## Diseño y Prototipado de Interfaces

Actualmente existen muchas herramientas de inteligencia artificial enfocadas en el **diseño de interfaces y prototipos**, que permiten generar diseños automáticamente a partir de texto o imágenes.

Algunas herramientas conocidas en esta categoría son:

- **Uizard** – Permite convertir bocetos o wireframes en interfaces digitales automáticamente mediante IA.
- **Galileo AI** – Genera interfaces completas para aplicaciones web o móviles a partir de descripciones en texto.
- **Figma AI** – Incorpora funciones de inteligencia artificial para generar layouts, componentes y sugerencias de diseño.
- **Framer AI** – Permite crear interfaces y páginas web completas a partir de prompts.
- **Locofy AI** – Convierte diseños de herramientas como Figma en código listo para desarrollo.
- **Visily AI** – Facilita la creación de wireframes y prototipos utilizando inteligencia artificial.

Estas herramientas ayudan a **acelerar el proceso de diseño**, permitiendo pasar de una idea a un prototipo funcional en pocos minutos. Sin embargo, **la mayoría ofrece versiones gratuitas con limitaciones o requiere planes de pago** para acceder a todas sus funciones.


## Stitch

Entre las herramientas disponibles, **Stitch** destaca por permitir generar interfaces modernas de forma rápida utilizando inteligencia artificial.

Stitch es una herramienta enfocada en el **Diseño y Prototipado de Interfaces**. Permite generar prototipos y diseños modernos a partir de **descripciones o prompts que indican el objetivo y la estructura de la interfaz**.

**Características:**

- Generación de interfaces desde **texto o imágenes**
- Genera diseños modernos utilizando **Tailwind**
- Permite crear interfaces para **aplicaciones web o móviles**
- Posibilidad de **exportar el prototipo con el código generado**

**Ejemplo de uso:**  
Mejorar un diseño antiguo de interfaz y transformarlo en uno más moderno utilizando IA, a partir de una descripción o cargando el diseño anterior.

## Programación asistida por IA

Actualmente existen herramientas que ayudan a los programadores a **escribir código más rápido, detectar errores y automatizar tareas repetitivas**.

Estas herramientas funcionan como **asistentes dentro del editor de código**, capaces de sugerir funciones completas, explicar código o incluso modificar proyectos completos.

### Herramientas principales

- **GitHub Copilot** – Asistente de programación que sugiere código automáticamente mientras el desarrollador escribe. Se integra con editores como Visual Studio Code.
- **Cursor** – Editor de código con inteligencia artificial integrada que puede generar, modificar y explicar código dentro del mismo proyecto.
- **Windsurf** – Editor de código con IA que ofrece autocompletado inteligente, generación de funciones y asistencia para múltiples lenguajes.
- **Trae** – Asistente de programación que ayuda a generar, optimizar y explicar código utilizando inteligencia artificial.
- **Kiro** – IDE con agentes de IA capaces de generar código, documentación y pruebas automáticamente.
- **Antigravity** – Editor de código experimental con agentes de IA que pueden planificar tareas, escribir código y ejecutar comandos dentro del entorno de desarrollo.

Estas herramientas permiten **acelerar el desarrollo de software**, reducir errores y mejorar la productividad de los programadores.

Sin embargo, muchas de estas plataformas ofrecen **versiones gratuitas con limitaciones**, como número de interacciones con IA, créditos mensuales o funciones restringidas. Para utilizar todas sus capacidades generalmente es necesario acceder a **planes de pago**.
